# Pokedex APP

##### Tecnologias

- Consumo da [ [PokéAPI ](https://pokeapi.co/)]()

- React-native com Expo Bare Workflow

- TypeScript

- Hooks

- React Navigation => StackNavigation e BottonTabNavigation

- Google fonts

- Axios

- SVG

- Lottie

- AsyncStorage (*Dependência instalada, ainda sem implementação*)
